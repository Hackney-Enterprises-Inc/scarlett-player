1. Simplest Drop-in (One Line!)

The easiest way to add a player - just add data-sp and src:

<!-- That's it! Just two attributes --> <div data-sp src="https://example.com/video.m3u8"></div> <!-- Or use a class --> <div class="scarlett-player" src="https://example.com/video.m3u8"></div> <!-- Or the full attribute name --> <div data-scarlett-player data-src="https://example.com/video.m3u8"></div>

2. Custom Branding (Multi-Tenant)

Player with custom brand colors - perfect for white-label streaming:

<div data-scarlett-player data-src="https://test-streams.mux.dev/x36xhzz/x36xhzz.m3u8" data-brand-color="#e50914" data-aspect-ratio="16:9" data-poster="https://via.placeholder.com/..." ></div>

3. Programmatic API

Create and control players with JavaScript:

const player = ScarlettPlayer.create({ container: '#programmatic-player', src: 'https://test-streams.mux.dev/x36xhzz/x36xhzz.m3u8', autoplay: false, brandColor: '#1e90ff', aspectRatio: '16:9', });

4. Auto-play (Muted)

Mobile-friendly auto-play with muted audio:

<div data-scarlett-player data-src="https://test-streams.mux.dev/x36xhzz/x36xhzz.m3u8" data-autoplay data-muted data-brand-color="#10b981" ></div>

5. Multiple Players

Different clients with different branding:

<!-- Client A: Red --> <div data-sp src="..." color="#e50914"></div> <!-- Client B: Blue --> <div data-sp src="..." color="#1e90ff"></div> <!-- Client C: Green --> <div data-sp src="..." color="#10b981"></div>

6. iframe Embed

For maximum isolation, use an iframe. Perfect for embedding on external sites:

<!-- Option 1: URL parameters --> <iframe src="https://embed.thestreamplatform.com/iframe.html?src=VIDEO_URL&brand-color=%23e50914" allow="autoplay; fullscreen" allowfullscreen ></iframe> <!-- Option 2: Video ID (requires Laravel backend) --> <iframe src="https://embed.thestreamplatform.com/v/abc123"></iframe> <!-- Option 3: Event slug --> <iframe src="https://embed.thestreamplatform.com/embed/fight-night-2025"></iframe>

How /v/abc123 works: The Laravel backend looks up the video by ID, fetches the stream URL and brand colors from the database, and returns a page with the player pre-configured. See templates/EmbedController.php for implementation.

7. Quick Copy Snippets

Copy-paste ready code for your site:

<!-- 1. Add the script (once per page) --> <script src="https://cdn.thestreamplatform.com/player/embed.umd.cjs"></script> <!-- 2. Add players anywhere --> <div data-sp src="YOUR_VIDEO_URL"></div>

Tip: Use color="#hex" to match your brand. Add autoplay muted for auto-playing videos.
